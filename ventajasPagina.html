Ventajas y desventajas de una página web

  1. Permite ofrecer información unificada y accesible.
  2. Es más fácil de mantener que un sitio web.
  3. Es más rápido y fácil navegar el contenido de una página web que de un sitio web.
  1. Al ser sólo una página, su estilo no cambia, lo que la vuelve monótona.
  2. A diferencia de un sitio web, una página web tiene poco espacio para mostrar elementos, por lo que podría sobrecargar la pantalla y dar una mal aspecto.
