Ventajas y desventajas de una página web

  1. Aprovecha al máximo las herramientas web disponibles.
  2. Son mucho más llamativas para los usuarios.
  3. Facilita la actualización del contenido, o inclusive, que sea creado por los mismos usuarios.
  1. Si no se tiene cuidado, podrían ser vulnerables a ataques cibernéticos.
  2. Si no está bien optimizada, podría arruinar la experiencia del usuario.
  3. Deben recibir actualizaciones constantes para arreglar bugs o mejorar las funcionalidades
